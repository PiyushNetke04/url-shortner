# URL Shortener

A full-stack URL Shortener built using Node.js, Express, MongoDB, and EJS.

## Features

- Convert long URL to short URL
- Custom short URL option
- Redirect to original link
- Click tracking
- Copy link button
- Delete link
- Stores data in MongoDB

## Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS
- nanoid

## How to run

npm install
node app.js

Open:
http://localhost:3000
